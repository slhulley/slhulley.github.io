Coffee Shop Website & Roasters Page

I didn’t have too much difficulty creating the roaster.html page. I started by creating a new HTML document and after typing the declaration and the following html and head tags I realized that the roasters page was basically the same as the coffee-shop.html page, so I copied and pasted the code into my new document. I then replaced the header image, the h1 text and the paragraph text and added the image after the closing p tag. Then I linked the new header image back to the homepage by enclosing the img tag in a link tag. I also used the same css document that we had previously created and linked it to the roaster.html document since there were no changes that need to be made in css.

I found that it was difficult to remember the correct format for links in html. To help myself in the future, I think I’ll do better if I re-read specific sections in the book every so often to refresh my memory.
